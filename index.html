Observer.js - a lightweight pub/sub.

Observer is a lightweight pub/sub library that can be used as a single event bus or as a base class for easy communications between objects.

https://github.com/jimenglish81/observer

To install run npm install then grunt.

Background

I authored Observer in order to have a lightweight pub/sub library to use while developing decoupled apps. This functionality is available through multiple libraries and frameworks, but I wanted to be able to have a library agnostic one.

Usage

To use Observer either create a new instance

var observerInstance = new Observer();

or use Observer's mixin functionality to decorate an object

var myObject = {};
var observerMixin = Observer.mixin(myObject);

Subscribing to an event

Subscribe via the subscribe method which takes two mandatory arguments and two optional ones.

  • eventName - required name of the event you wish to subscribe to.
  • handler - required handler you wish to run when the event is published.
  • scope - optional the scope you wish the supplied handler to run in.
  • once - optional if you wish the handler to only be triggered once and then immediately unsubscribed. 
var observerInstance = new Observer();
observerInstance.subscribe('myEvent', function() {
   console.log('event published!');
});

Publishing an event

Publish via the publish method which takes one mandatory and any number of other arguments which will be proxied on to the handler.

  • eventName - required name of the event you wish to publish.
var observerInstance = new Observer();
observerInstance.subscribe('myEvent', function() {
   console.log(myArg1, myArg2); // will log "'foo' 'bar'"
});
observerInstance.publish('myEvent', 'foo', 'bar');

Unsubscribing from an event

Subscribe via the unsubscribe method which takes two optional arguments. If neither is supplied it will unsubscribe from all events.

  • eventName - optional name of the event you wish to unsubscribe from.
  • scope - optional scope of you wish to unsubscribe from.
var observerInstance = new Observer();
observerInstance.unsubscribe('myEvent'); // unsubscribe to myEvent for all scopes.
observerInstance.unsubscribe('myEvent', window); unsubscribe to myEvent for handlers in the window scope.
observerInstance.unsubscribe(null, window); unsubscribe to all events for handlers in the window scope.
observerInstance.unsubscribe(); // unsubscribe to all events in all scopes
